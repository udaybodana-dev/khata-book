# 💸 Khata Book

A comprehensive financial ledger application. Manage mathematical states, track custom categories, and automatically balance income against expenses in a clean, high-performance interface.

`Documentation Update (2026): Added live deployment link -` https://udaybodana-dev.github.io/khata-book/

---

## ✨ Features

- **Dynamic Ledger Management:** Easily record, track, and categorize custom transactions as Income or Expense.
- **Smart Summaries:** Real-time automatic computation of Net Balance, Total Income, and Total Expense.
- **Color-Coded History:** Transaction history logs are beautifully styled with visual type markers and deletion options.
- **Persistent Local Storage:** Retains your full history and ledger states between browser sessions using `localStorage`.

---

## 🚀 How to Use

1. **Add Transaction:** Fill out the transaction name, enter the amount, select its category, and specify its type (Income or Expense).
2. **Review Summary:** Watch your Net Balance, Total Income, and Total Expenses update instantly in real time.
3. **Delete Entries:** Use the delete buttons to cleanly remove transactions from the historical ledger list.
